Travail de Maturité (Lycée Denis-de-Rougemont - 2017)

1.  Ce jeu est utilisable uniquement sur les navigateurs FIREFOX et CHROME
2.  Pour lancer le programme, il faut ouvrir le fichier "index.html"
3.  Pour naviguer dans le menu du jeu, il faut utiliser les flèches du clavier
4.  Pour déplacer le personnage, il faut également utiliser les touches "WASD"
5.  Pour attaquer lors d'un combat, il faut appuyer le plus de fois possible sur la barre SPACE
6.  Pour soigner sa créature, il faut parler à la vieille dame qui se trouve vers les fontaines
